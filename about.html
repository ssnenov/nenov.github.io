Hello, my name is Simeon Nenov and thank you for visiting my blog.

I am Principal Software Engineer at Progress Software where we are developing mostly web applications using .NET technologies and frameworks such as ASP.NET WebForms, MVC and Web API, Entity Framework, Telerik DataAccess, Sitefinity CMS and ServiceStack. Obviously I'm a big fan of .NET/.NET core and everything related and I'm very passionate to talk about it.

In my spear time I like to play volleyball, skiing, sauna and write code for fun. When coding for my self I like to play with things like .NET Compiler Platform (Roslyn), Visual Studio extensions, homemade IoC containers with AOP capabilities, handmade performance profilers and low level things like MSIL instrumentation, code weaving, hardcore reflection and expression trees. In other words I like to write frameworks/libraries by my self (just like this song: we are gonna build a framework 'cos we wanna use one, but don't wanna choose one).

I hope you have a great time with my blog and learn something new or I helped you in your day-to-day work.
